Add Triple Crown Trek Planner to Your Home Screen

Walk With Others · Free Forever · triplecrowntrekplanner.com
Installing puts a real app icon on your phone or computer. It launches full-screen (no browser bars), works offline — every trail, profile, and waypoint is built in, so it runs with no signal — and there is no app store, no account, and no fee. Takes about ten seconds.

On an iPhone or iPad

Use Safari
Share → Add to Home Screen
  1. Open triplecrowntrekplanner.com in Safari (this only works in Safari, not Chrome, on iPhone).
  2. Tap the Share button — the square with an arrow pointing up — in the toolbar.
  3. Scroll down and tap “Add to Home Screen.”
  4. Tap Add in the top corner. The green mountain icon appears on your home screen.
Already tried once and got a blank icon? Delete that one, then add it again — iPhones cache the old icon hard.

On an Android phone

Use Chrome
Menu (⋮) → Install app
  1. Open triplecrowntrekplanner.com in Chrome.
  2. Tap the menu — the three dots in the top-right corner.
  3. Tap “Install app” (some phones say “Add to Home screen”).
  4. Confirm with Install / Add. The icon lands on your home screen.
If a banner pops up at the bottom offering to install, you can just tap that instead — same result.

On a computer

Chrome or Edge
Install icon in the address bar
  1. Open triplecrowntrekplanner.com in Chrome or Edge.
  2. Look at the right end of the address bar for the small install icon (a monitor with a down-arrow). Click it.
  3. Click Install in the little box that appears.
  4. The app opens in its own window and lands in your Applications / Launchpad (Mac) or Start menu (Windows).
Tip (Mac): right-click the app’s Dock icon → Options → Keep in Dock to pin it. A bookmark or desktop link is not the same as installing — it won’t carry the icon or run offline.
